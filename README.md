## Notes

- `perKwh` values are in **cents per kWh**.
- `nemTime` is the interval timestamp returned by Amber.
- This README is regenerated on every run, so the latest 24-hour table replaces the previous table.
- The accumulated JSON files keep historical intervals and append only new records.
